MCP מהטרמינל

Claude Code מגיע עם ממשק CLI מלא לניהול שרתי MCP — מוסיפים, מסירים ובודקים חיבורים בלי לגעת בממשק גרפי.

כל ההגדרות נשמרות בקובץ JSON שניתן לגרסן, לשתף ולשכפל — MCP כחלק מה-infrastructure של הפרויקט.

claude mcp · settings.json · scope

פקודות בסיסיות

1add

הוספת שרת MCP

מוסיפים שרת MCP לפי שם הפקודה שלו. Claude Code ישאל על הגדרות נוספות כמו API key אם צריך.

$ claude mcp add linear
2list

רשימת השרתים

מציג את כל שרתי ה-MCP המוגדרים — שם, status חיבור וה-scope שלהם (global / project).

$ claude mcp list
3remove

הסרת שרת

מסיר שרת MCP לפי שמו. השינוי מיידי — Claude Code לא יוכל יותר להשתמש בכלים של אותו שרת.

$ claude mcp remove linear

Scope — גלובלי ומקומי

1global

Global — לכל הפרויקטים

ברירת המחדל. השרת זמין בכל תיקייה שנפתח בה Claude Code. מתאים לכלים כמו Linear, Notion — שמשמשים בכל פרויקט.

$ claude mcp add --global linear
~/.claude/settings.json
2project

Project — לפרויקט ספציפי

השרת מוגדר רק בתיקיית הפרויקט הנוכחי. מתאים לכלים ייעודיים — בדיקות, DB, שירות ספציפי.

$ claude mcp add --project playwright
.claude/settings.json

בתוך הסשן

1status

בדיקת חיבור

בתוך Claude Code מריצים /mcp כדי לראות אילו שרתים מחוברים ואילו כלים זמינים בסשן הנוכחי.

בתוך Claude Code → הקלידו /mcp
2get

פרטי שרת

מציג את ההגדרות המלאות של שרת ספציפי — command, env vars, status. שימושי לדיבאג.

$ claude mcp get linear
3reset

אתחול חיבור

אם שרת MCP תקוע, מריצים reset כדי לסגור ולפתוח מחדש את החיבור — בלי לצאת מהסשן.

$ claude mcp reset linear
