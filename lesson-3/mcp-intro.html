מה זה MCP?

MCP הוא פרוטוקול תקני שקובע איך AI מדבר עם כלים חיצוניים.

כמו ש-HTTP קובע איך דפדפנים מדברים עם שרתים — MCP קובע איך Claude מחובר לכלים חיצוניים.

anthropic · open standard · 2024

הארכיטקטורה

1ai

Claude Codeמקבל ההחלטות

מחליט מתי לקרוא לכלי ואיך להשתמש בתשובה. שולח בקשות ל-MCP server ומשלב את התוצאות בעבודה.

2mcp server

MCP Serverהמתאם

תוכנית קטנה שחושפת יכולות של כלי אחד ומדברת עם Claude בפרוטוקול שהוא מבין. לכל כלי יש server משלו.

3protocol

JSON-RPCשפה משותפת

פרוטוקול אחיד לכל שפת תכנות. מאפשר לחבר כלים שונים עם Claude בלי לדעת איך כל אחד בנוי מבפנים.

סוגי יכולות

1tools

כלי פעולה

Claude מפעיל פעולות בשמנו — הוא מחליט מתי ואיך לפי הבקשה שנתנו לו.

  • · Playwright — מנווט, לוחץ, ממלא טפסים, מצלם מסך
  • · Linear — פותח tasks, מעדכן status, מחפש בפרויקט
  • · GitHub — פותח PR, קורא issues, מחפש קוד
2resources

קריאת מידע

Claude שואב מידע ישירות מהמקור — בלי שנצטרך להעתיק ולהדביק לצ'אט.

  • · Filesystem — קורא קבצים מהמחשב שלנו
  • · Context7 — שולף תיעוד עדכני של כל ספרייה
  • · Linear — טוען issue עם כל ההיסטוריה שלו
3prompts

פקודות מוכנות

ה-server מגדיר פקודות מוכנות מראש. שם אחד מפעיל רצף פעולות שלם.

  • · Playwright — "כתוב בדיקה E2E לפיצ'ר"
  • · Linear — "פתח task לבאג שמצאת"
  • · GitHub — "סכם את ה-PR הפתוחים"

שרתים פופולריים

1github

GitHub MCP

הכי מותקן — issues, PRs וחיפוש קוד בכל ה-repos. דורש GITHUB_PERSONAL_ACCESS_TOKEN.

› claude mcp add github -- npx -y @modelcontextprotocol/server-github
2context7

Context7

הכי פופולרי — מזריק תיעוד עדכני ל-100+ frameworks ישירות לתוך ה-context של Claude.

› claude mcp add context7 -- npx -y @upstash/context7-mcp
3playwright

Playwright MCP

שליטה מלאה בדפדפן — ניווט, לחיצות, צילומי מסך. ה-AI הופך לאוטומציה של דפדפן אמיתי.

› claude mcp add playwright -- npx @playwright/mcp@latest
4linear

Linear MCP

ניהול issues, cycles ו-projects ב-Linear. גישה דרך HTTP — ללא התקנה מקומית.

› claude mcp add --transport http linear https://mcp.linear.app/mcp
