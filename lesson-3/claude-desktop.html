מה זה Claude Desktop?

אפליקציה מקומית ל-Mac + Windows — לא עוד טאב בדפדפן. גישה ישירה לקבצים, שלושה מצבי עבודה, ו-50+ Connectors.

היתרון הגדול: Claude יכול לגשת למחשב שלנו, לקרוא קבצים, ולפעול אוטונומית — בלי שנצטרך להדביק לצ'אט.

MCP · Connector · Plugin

MCP

מתווסף ידנית דרך Developer Settings — דורש עריכת קובץ הגדרות וחיפוש עצמאי של ה-MCP server.

Connector

דרך קלה יותר להוסיף MCP server מתוך ממשק Claude Desktop. עדיין יכול להידרש אימות בחלק מהשרתים — אבל ללא עריכת קבצי הגדרות.

Plugin

מכיל גם Connector וגם Skills רלוונטיים — חבילה מוכנה עם חיבור לשירות ויכולות AI מותאמות לאותו שירות.
