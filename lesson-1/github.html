מה זה GitHub?

GitHub היא פלטפורמת ענן לאחסון ושיתוף קוד. היא לא Git — היא הבית שבו הפרויקטים שלנו חיים ברשת ומאפשרת לצוות לעבוד יחד על אותו קוד.

כל שינוי, כל גרסה, כל שיתוף — נשמרים ומסונכרנים אוטומטית.

מושגי ליבה

1 repo

Repository

כל הקבצים, ההיסטוריה וה-commits של הפרויקט — כולם בתיקייה אחת, מסונכרנת לענן.

2 branch

Branch

עובדים על פיצ'ר חדש בנפרד מהקוד הראשי — בלי לשבור את מה שעובד.

3 fork

Fork

עותק אישי של repository של מישהו אחר — הבסיס לתרומה לפרויקטי קוד פתוח.

4 PR

Pull Request

בקשה לאחד שינויים מ-branch חזרה ל-main — כולל אפשרות ל-code review.

יוצרים ומשכפלים

1 create

יוצרים Repository

ב-GitHub לוחצים New → מזינים שם לפרויקט → בוחרים Public או Private → לוחצים Create repository.

2 clone

משכפלים למחשב

לוחצים על Code ב-GitHub, מעתיקים את ה-URL, ומריצים git clone בטרמינל — Git יוצר תיקייה מקומית עם כל הקוד.

3 open

פותחים בעורך הקוד

נכנסים לתיקייה שנוצרה ופותחים אותה ב-VS Code — מוכנים להתחיל לעבוד.

Pull Request

1 branch

עובדים על ענף

יוצרים ענף חדש לכל פיצ'ר או תיקון — כך ה-main נשאר יציב ונקי, ואפשר לעבוד במקביל.

2 push

שולחים לענן

אחרי commits, שולחים את הענף ל-GitHub — הוא מופיע שם ואחרים יכולים לראות ולהעיר.

3 open PR

פותחים Pull Request

ב-GitHub לוחצים Compare & pull request, מוסיפים תיאור ושולחים לעיון לפני מיזוג.
