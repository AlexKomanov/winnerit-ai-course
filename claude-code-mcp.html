MCP מהטרמינל

Claude Code כולל ממשק CLI מלא לניהול MCP servers — מוסיפים כלי חדש בפקודה אחת, ו-Claude Code מיד יכול להשתמש בו.

ההגדרות נשמרות בקובץ JSON — ניתן לשמור אותו ב-Git ולשתף עם הצוות.

claude mcp add · list · remove · get

הוספת MCP Server

1add · local

server מקומי

רץ על המחשב שלנו. Claude Code מפעיל אותו אוטומטית — לדוגמה Playwright.

› claude mcp add playwright npx @playwright/mcp@latest
2add · http

server מרוחק

רץ בענן. נדרש להוסיף את ה-URL עם --transport http.
לדוגמה: Linear.

› claude mcp add --transport http linear https://mcp.linear.app/mcp
3status · remove

ניהול servers

בדיקת servers פעילים בסשן הנוכחי, והסרה לפי שם מהטרמינל.

› /mcp
› /mcp remove playwright

Scope — גלובלי ומקומי

1user · global

User — לכל הפרויקטים

ברירת המחדל. ה-server זמין בכל תיקייה שפותחים בה Claude Code. מתאים לכלים כמו Playwright, Linear — שמשמשים בכל פרויקט.

› claude mcp add --scope user playwright ...
~/.claude/settings.json
2project · local

Project — לפרויקט ספציפי

ה-server מוגדר רק בפרויקט הנוכחי. קובץ ה-JSON ניתן ל-commit ב-Git — כל הצוות מקבל את אותן ההגדרות.

› claude mcp add --scope project playwright ...
.mcp.json
project scope — general example
{
  "mcpServers": {
    "shared-server": {
      "command": "/path/to/server",
      "args":    [],
      "env":     {}
    }
  }
}
user scope — playwright mcp example
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": [
        "@playwright/mcp@latest"
      ]
    }
  }
}
