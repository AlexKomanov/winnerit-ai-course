מה זה agent?

LLM שיכול לקבל החלטות, להשתמש בכלים חיצוניים, ולבצע רצף של פעולות — עד שהמשימה הושלמה.

הוא לא רק עונה על שאלה. הוא פועל.

agent session
user › מי זכה בגביע העולם ב־2022?
🔧 web_search("2022 FIFA World Cup winner")
🔍 result: Argentina defeated France...
agent › ארגנטינה זכתה ב־2022
בגמר נגד צרפת, 4:2 בפנדלים.

vs

Agent

multi-step
  • +מתכנן רצף של צעדים
  • +שומר זיכרון לאורך המשימה
  • +קורא לכלים: חיפוש, קוד, קבצים
  • +יכול לפעול באופן אוטונומי

הכלים של הסוכן

חיפוש רשת

שליחת שאילתה למנוע חיפוש וקבלת תוצאות בזמן אמת. הסוכן מקבל מידע עדכני שאינו נמצא בתוך המודל.

הרצת קוד

כתיבה והרצה של קוד Python בתוך sandbox. הסוכן מחשב, מנתח נתונים ומייצר גרפים באופן אוטונומי.

מערכת קבצים

קריאה, כתיבה ועריכה של קבצים. הסוכן מנתח מסמכים, שומר תוצאות ומנהל מידע לאורך זמן.

01/03

חשיבה רב-שלבית

הסוכן מפרק כל משימה מורכבת לצעדים קטנים, מתכנן לפני הביצוע ומתאים את עצמו לכל תוצאה שמתקבלת. בניגוד ל-chat — הוא לא עוצר אחרי תשובה אחת.

02/03

פעולה בעולם האמיתי

הסוכן לא מוגבל לטקסט — הוא מחפש, מחשב, קורא קבצים ושולח בקשות API. המודל הופך ממנוע שפה לעוזר שפועל.

03/03

זיכרון לאורך המשימה

הסוכן שומר את כל ההקשר — כל כלי שקרא, כל תוצאה שקיבל — ומשתמש בו בצעד הבא. כל צעד מתבסס על מה שנלמד לפניו.

Agents + Ollama

רוב ה-frameworks תומכים ב-OpenAI API — ו-Ollama חושף בדיוק אותו API. סוכן מקומי לחלוטין: ללא ענן, ללא עלות, ללא פגיעה בפרטיות.

נשתמש ב-smolagents עם gemma4 דרך Ollama.

local agent · ollama
$ ollama pull gemma4
pulling manifest ...
✓ model ready (4.7 GB)
$ python agent.py
🔧 search("אוכלוסיית ישראל 2025")
🔍 ~9.9 million...
agent › כ-9.9 מיליון תושבים.

מגדירים את הסביבה

1install

מתקינים smolagents

ספריית הסוכנים של HuggingFace — קומפקטית ותומכת ב-Ollama באמצעות OpenAI-compatible API.

$ pip install smolagents
2connect

מחברים ל-Ollama

מגדירים את המודל המקומי כ-backend של הסוכן — Ollama רץ על localhost:11434.

› from smolagents import OpenAIServerModel
3define tools

מגדירים כלים

כל כלי הוא פונקציה Python עם docstring — הסוכן לומד ממנה מתי ואיך להשתמש בו.

› from smolagents import tool

סיכונים שצריך להכיר

הלוצינציות בכלים

הסוכן עשוי לקרוא לכלי עם פרמטרים שגויים, להמציא תוצאות, או להסיק מסקנות מוטעות מהנתונים שקיבל.

לולאות אינסופיות

ללא הגדרת max_steps הסוכן יכול להיתקע בלולאה ולבזבז משאבים. תמיד מגדירים מגבלת צעדים.

הרשאות-יתר

סוכן עם גישה לקבצים, DB או API יכול לגרום נזק בלתי הפיך. נותנים רק הרשאות מינימליות הכרחיות.
