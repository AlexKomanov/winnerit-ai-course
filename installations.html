GitHub

נפתח חשבון — זה המקום בו נשמור ונשתף את הקוד שלנו לאורך כל הקורס.

1go to site

כניסה לאתר

נגיע לדף ההרשמה של GitHub. נבחר שם משתמש באנגלית — הוא יהיה חלק מהקישורים לפרויקטים שלנו.

→ github.com/signup
2fill form

מילוי הפרטים

נמלא: כתובת אימייל, סיסמה, שם משתמש. לאחר מכן פתרון Captcha קצר, ולחיצה על Create account.

→ Email · Password · Username → Create account
3verify email

אימות האימייל

GitHub ישלח קוד אימות לאימייל שהזנו. נזין את הקוד — החשבון מוכן.

→ Check email → Enter 6-digit code

Git

מערכת ניהול גרסאות — נבחר בדרך המתאימה למערכת שלנו.

macOS · Homebrew

macOS — דרך brew

פקודה אחת בטרמינל. אם Homebrew עדיין לא מותקן — נתקין אותו תחילה מ-brew.sh.

>_ brew install git
Windows · Installer

Windows — installer

נוריד את קובץ ה-.exe מהאתר הרשמי ונריץ. נשאיר את כל ברירות המחדל — Git ייכנס אוטומטית ל-PATH.

→ git-scm.com/download/win
macOS · Windows
1verify

אימות ההתקנה

נפתח טרמינל חדש ונריץ. אם מוצג משהו כמו git version 2.x.x — ההתקנה הצליחה.

>_ git --version

Git · הגדרה

נגדיר שם ואימייל — חובה להשתמש באותם פרטים כמו חשבון GitHub. כל commit יישמר עם הפרטים האלה.

macOS · Windows
1set name

הגדרת שם

נגדיר את השם שיופיע ב-commits. נשתמש בשם האמיתי שלנו — זה מה שיראו שותפים לפרויקט.

>_ git config --global user.name "Your Name"
macOS · Windows
2set email

הגדרת אימייל

נגדיר את האימייל — חייב להיות זהה לאימייל של חשבון GitHub שפתחנו. זה מחבר בין ה-commits לפרופיל.

>_ git config --global user.email "you@..."

Node.js

סביבת הרצה ל-JavaScript — נבחר בדרך המתאימה לנו, npm מגיע אוטומטית עם ה-Node.

macOS · Installer / brew

macOS — installer

נוריד מהאתר הרשמי, נבחר LTS ונריץ את ה-.pkg. דרך Homebrew אפשרית גם: brew install node

→ nodejs.org/en/download → LTS → .pkg
Windows · Installer

Windows — installer

נוריד את ה-LTS מהאתר הרשמי — קובץ .msi. נריץ, נשאיר הכל כברירת מחדל, ו-"Add to PATH" יסומן אוטומטית.

→ nodejs.org/en/download → LTS → .msi
macOS · Windows
1verify

אימות ההתקנה

נריץ שתי פקודות בטרמינל — אם מוצגות גרסאות, הכל מותקן בהצלחה. Node ו-npm מוכנים לשימוש.

>_ node -v && npm -v

VS Code Insiders

גרסת הפיתוח של VS Code — עדכונים מוקדמים ותמיכה מקסימלית בתוסף Claude. נוריד ונתקין.

macOS · Windows
1download insiders

הורדת VS Code Insiders

נגיע לדף ההורדות ונבחר את הגרסה למערכת ההפעלה שלנו. נריץ את המתקין ונשאיר ברירות מחדל.

→ insiders.vscode.com/download
macOS · Windows
2first launch

הפעלה ראשונה

נפתח את VS Code Insiders ונוודא שהעורך נטען בהצלחה. הסמל של Insiders הוא ירוק — שונה מ-VS Code הרגיל.

→ Open VS Code Insiders ✓
שים לב · already have an editor? יש לכם כבר VS Code, Cursor, או Antigravity? אפשר להישאר איתם — התוסף של Claude עובד גם שם. VS Code Insiders מומלץ כנקודת התחלה נקייה.

תוסף Claude

נתקין את תוסף Claude בתוך VS Code Insiders — האינטגרציה בין העורך לבין ה-AI.

macOS · Windows
1open extensions

פתיחת Extensions

נפתח את חלונית ה-Extensions בעזרת קיצור המקשים המתאים למערכת.

→ ⌘+Shift+X (mac) · Ctrl+Shift+X (win)
macOS · Windows
2install extension

חיפוש והתקנה

נחפש בשורת החיפוש את שם התוסף המדויק. נמצא את התוסף של Anthropic ונלחץ Install.

→ Search "claude code for vscode" → Install
Extensions — VS Code Insiders

Claude · חשבון

נירשם לחשבון Claude ונשדרג ל-Pro — מנוי שמאפשר גישה לכלים מתקדמים, כולל Claude Code CLI.

1sign up

יצירת חשבון

נגיע לאתר ונירשם עם כתובת אימייל או דרך חשבון Google / Apple. נאמת את הכתובת בהודעה שנקבל.

→ claude.ai → Sign up
2upgrade pro

שדרוג ל-Pro

מתוך Settings → Billing, נבחר ב-Pro. המנוי כולל גישה למודלים החזקים ביותר ולימיט גבוה יותר.

→ Settings → Billing → Upgrade to Pro
3first chat

שיחה ראשונה

נוודא שהחשבון עובד — נפתח שיחה חדשה ונשאל משהו פשוט. ניתן להשאיר את המודל כברירת מחדל, או לבחור Claude Sonnet 4.5 ומעלה.

→ New chat → "שלום!" ✓

Claude Desktop

אפליקציית Desktop מאפשרת לחבר את Claude לקבצים, כלים ומחשב שלנו — מעבר לדפדפן.

macOS · Windows
1download

הורדת Desktop

נגיע לדף ההורדות ונבחר את הגרסה למערכת שלנו. נריץ את המתקין ונאשר את ההרשאות הנדרשות.

→ claude.ai/download
macOS · Windows
2sign in

התחברות לחשבון

נפתח את האפליקציה ונתחבר עם חשבון Claude שיצרנו. הדפדפן ייפתח לאישור — נאשר ונחזור לאפליקציה.

→ Open Claude Desktop → Sign in → Authorize

Claude Code CLI

כלי שורת הפקודה של Claude — נריץ את סקריפט ההתקנה הרשמי לפלטפורמה שלנו ונתחבר לחשבון.

macOS · curl script
1macos install

macOS — התקנה

נריץ את סקריפט ההתקנה הרשמי בטרמינל. הסקריפט יתקין את Claude Code ויוסיף אותו ל-PATH אוטומטית.

>_ curl -fsSL https://claude.ai/install.sh | sh
Windows · PowerShell
1windows install

Windows — התקנה

הסקריפט יוריד וייתקין אוטומטית — הכלי יהיה זמין ב-Terminal לאחר ההתקנה.

>_ iwr -useb https://claude.ai/install.ps1 | iex
macOS · Windows
2login

התחברות ראשונה

נריץ בטרמינל — הדפדפן ייפתח לאישור גישה לחשבון Claude שלנו. תהליך חד-פעמי, לאחריו Claude Code מוכן.

>_ claude login

Ollama

כלי להרצת מודלי AI מקומית — נבחר את דרך ההתקנה המתאימה למערכת שלנו.

macOS · curl / dmg

macOS — Terminal

הסקריפט מזהה את המארכיטקטורה (Apple Silicon / Intel) ומתקין אוטומטית. חלופה גרפית: .dmg מ-ollama.com/download

>_ curl -fsSL https://ollama.com/install.sh | sh
Windows · PowerShell / exe

Windows — PowerShell

הסקריפט מתקין Ollama אוטומטית. חלופה גרפית: קובץ .exe מ-ollama.com/download

>_ irm https://ollama.com/install.ps1 | iex

Ollama · הרצה

נוריד מודל מקומי ונשוחח איתו — הורדה חד-פעמית, לאחריה עובד offline לחלוטין.

macOS · Windows
1pull model

הורדת מודל

נוריד את gemma4 — מודל קל ומהיר של Google. כ-2GB, הורדה חד-פעמית. תוצגנה שורות התקדמות.

>_ ollama pull gemma4
macOS · Windows
2run & chat

הרצה ושיחה

נריץ ונתחיל שיחה ישירות בטרמינל. לצאת: /bye. המודל עובד לגמרי מקומית, ללא רשת.

>_ ollama run gemma4

1sign up

יצירת חשבון

נגיע לאתר ונירשם דרך חשבון Google — כניסה מהירה ללא צורך בסיסמה נפרדת. ניתן גם להירשם עם אימייל.

→ lovable.dev → Continue with Google

→ Dashboard → Explore UI
3first project

→ New Project → enter a prompt → Generate
