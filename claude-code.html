Claude Code
AI שמבין את הקוד שלנו.

כלי command-line של Anthropic שמחבר בין Claude לפרויקט. נתאר מה נרצה — בשפה טבעית — והוא יקרא קבצים, יכתוב קוד, יריץ פקודות ויתקן באגים. ישירות בטרמינל, בלי GUI, בלי plugin.

~/my-project zsh
➜ claude
 
> Add logging to all functions in utils.py
✔ reading utils.py…
✔ editing utils.py
Added logging to 6 functions.
 
>

איך זה עובד?

קורא את הפרויקט

לפני שכותב שורה אחת — Claude Code סורק קבצים, תיקיות, git history ו-CLAUDE.md. הוא מבין את הבסיס לפני שמתחיל לפעול.

שפה טבעית

מתארים מה נרצה — בעברית או אנגלית, בלי תחביר מיוחד. "תוסיף בדיקות ל-UserService" או "תתקן את הבאג בשורה 42" — זה מספיק.

כותב ועורך קוד

יוצר קבצים חדשים, עורך קוד קיים, מוחק מה שמיותר. כל שינוי מוצג לאישורנו לפני הביצוע — שום דבר לא קורה מאחורי הגב.

מריץ פקודות

עובד ישירות עם הטרמינל — מריץ bash, npm, git, pytest ועוד. קורא את הפלט, ומגיב בהתאם. לא צריך להעתיק-הדביק.

לולאת תיקון

אם test נכשל — Claude Code רואה את השגיאה, מנתח, ומתקן. ממשיך בלולאה עד שהבדיקות עוברות. אנחנו רק מאשרים בסוף.

Anthropic API

ה"חשיבה" של Claude מתבצעת בענן דרך ה-API. הכלים עצמם — קריאה, כתיבה, הרצה — מבוצעים מקומית על המחשב שלנו בלבד.

נתקין את Claude Code

פקודה אחת — בוחרים לפי המערכת.

1option · macos / linux

macOS · Linux · WSL

ההתקנה המומלצת. מתעדכנת אוטומטית ברקע לגרסה החדשה ביותר.

>_ curl -fsSL https://claude.ai/install.sh | bash
2option · homebrew

Homebrew

חלופה למשתמשי Mac שמעדיפים Homebrew. לא מתעדכן אוטומטית — נריץ brew upgrade ידנית.

>_ brew install --cask claude-code
3option · windows

Windows PowerShell

נפתח PowerShell ונריץ את הפקודה. למשתמשי CMD ו-WinGet יש הוראות נפרדות בתיעוד.

>_ irm https://claude.ai/install.ps1 | iex
הוראות מלאות, desktop app ו-VS Code extension — code.claude.com/docs code.claude.com/docs

נתחבר ל-Claude

בהרצה הראשונה נתבקש להתחבר — פעם אחת. לאחר מכן הפרטים נשמרים אוטומטית.

1step · run

נריץ claude

ניכנס לתיקיית הפרויקט ונריץ. בהרצה הראשונה Claude Code יזהה שאין חיבור ויפנה אותנו להתחברות.

>_ claude
2step · browser

נאשר בדפדפן

נפתח חלון דפדפן — נכנסים עם חשבון Claude.ai (Pro / Max / Team / Enterprise). אפשר גם להתחבר מחדש מתי שנרצה.

>_ /login

✦  אין מינוי? אפשר גם להתחבר דרך Anthropic Console עם API Key — תשלום לפי שימוש ללא מינוי חודשי.

CLAUDE.md

קובץ markdown בשורש הפרויקט. Claude Code קורא אותו אוטומטית בכל שיחה — ההנחיות, האדריכלות והמוסכמות שנכתוב שם יחסכו הסברים חוזרים.

נרשום את סטק הטכנולוגיות ואת מבנה הפרויקט

מערכת ההרשאות

קריאת קבצים

Claude Code קורא קבצים בפרויקט באופן חופשי — זה בטוח ולא דורש אישור מפורש שלנו.

עריכת קבצים

לפני כל שינוי — Claude מציג את ה-diff ומחכה לאישורנו. נוכל לאשר, לדחות או לבקש שינוי.

פקודות Bash

הרצת פקודות בטרמינל — כולל git, npm ו-scripts — דורשת אישור מפורש שלנו לפני הביצוע.

MCP — כלים נוספים

מה זה MCP?

Model Context Protocol — תקן שמאפשר ל-Claude Code להתחבר לכלים חיצוניים: דפדפן, GitHub, DB ועוד.

הוספת שרת MCP

נוסיף שרת MCP דרך הגדרות Claude Code. מאות integrations זמינים — ואפשר לבנות שרת מותאם.

דוגמאות

Playwright (דפדפן), GitHub, Notion, Google Drive, Supabase, Canva — וכלים מותאמים אישית.

טיפים לשימוש יעיל

תאר את ה-context

נכתוב ב-CLAUDE.md את הסטק, האדריכלות והמוסכמות. ככה Claude לא מנחש — הוא יודע.

בקש תוכנית קודם

לפני שינויים גדולים — נבקש "write a plan first". נבחן ונגיב לפני שיתחיל לממש.

שמור שיחות קצרות

אחרי כל משימה — /clear. שיחות קצרות = context נקי = תוצאות טובות יותר.
