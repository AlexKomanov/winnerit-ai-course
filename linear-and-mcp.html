מה זה Linear?

Linear הוא כלי ניהול פרויקטים מודרני שבנוי בשביל מפתחים — מהיר, נקי, ומכוון לעבודה אמיתית.

במקום טפסים ארוכים ו-workflows מסורבלים, Linear מדגיש מהירות ופשטות — issue נפתח תוך שניות ו-backlog תמיד מסודר.

linear.app · fast · keyboard-first

מה זה Linear MCP?

Linear MCP הוא שרת MCP שמחבר את Claude Code ישירות ל-Linear — ללא צורך לעבור בין חלונות.

מתארים בשפה טבעית מה צריך לעשות — ה-AI פותח issue מסודר עם כותרת, תיאור, priority ו-labels.

linear.app · mcp · claude code integration

התקנה וחיבור

1api key

יוצרים API Key

ב-Linear נייצר מפתח API אישי שייתן ל-Claude Code הרשאות לנהל issues בשמנו.

Linear → Settings → API → Personal API keys → Create key
2install

מוסיפים ל-Claude Code

מפעילים את פקודת ה-MCP ומוסיפים את Linear MCP server עם ה-API key שיצרנו.

› /mcp add linear
3verify

בדיקת חיבור

נבקש מ-Claude Code לרשום את ה-issues האחרונים שלנו — אם ניתן רשימה, החיבור עובד.

› show my recent Linear issues

פעולות בסיסיות

1create

פתיחת Issue

מתארים בעברית או באנגלית — Claude Code יפתח issue מסודר עם acceptance criteria ו-labels.

› create a Linear issue: login button unresponsive on mobile, high priority, team Frontend
2search

חיפוש ב-Backlog

שואלים שאלות טבעיות על ה-backlog — מה לא נגמר, מה בעדיפות גבוהה, מה לא הוקצה לאף אחד.

› list all open high-priority issues not assigned to anyone
3update

עדכון Status

מזכירים ל-Claude Code את מספר ה-issue ומבקשים לעדכן status, priority או assignee.

› mark issue ABC-42 as In Progress and assign to me

Workflow מלא

1from bug to issue

מבאג ל-Issue

מוצאים בעיה בקוד — מתארים ל-Claude Code בצ'אט. הוא מנסח issue מפורט עם steps to reproduce, expected behavior ו-impact, ופותח אותו ב-Linear.

› create a bug report for what I just found: users can't log in after password reset
2from prd to backlog

מ-PRD ל-Backlog

מדביקים PRD בצ'אט ומבקשים מ-Claude Code לפרק אותו לרשימת issues ב-Linear. תוך שניות יש backlog מסודר לפי feature, priority ו-team.

› read this PRD and create Linear issues for each feature, grouped by component
