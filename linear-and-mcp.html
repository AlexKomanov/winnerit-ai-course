מה זה Linear?

Linear הוא כלי לניהול משימות ופרויקטים שבנוי למפתחים — מהיר ונקי.

אם עבדתם עם Jira — הרעיון מוכר: issues, sprints, backlog. רק ב-Linear הכל קורה הרבה יותר מהר — ממשק נקי ומהיר.

מה זה Linear MCP?

Linear MCP הוא שרת MCP שמחבר את Claude Code ישירות ל-Linear — ללא צורך לעבור בין חלונות.

כותבים בשפה טבעית מה לעשות — ה-AI פותח issue מסודר עם כותרת, תיאור, priority ו-labels, מחפש ב-backlog, מעדכן status — הכל מהטרמינל.

mcp.linear.app · http transport · natural language

שלוף מה-PRD את כל המידע על הפיצ'ר "כניסה עם Google".
1. פתח issues ב-Linear לכל משימת פיתוח
2. כתוב Playwright tests לפי ה-acceptance criteria ושמור אותם בתיקיית tests/
3. הרץ את ה-tests עם Playwright MCP
4. עדכן את ה-status של כל issue לפי תוצאות הבדיקות
