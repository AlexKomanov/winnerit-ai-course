Ollama — מה זה?

כלי קוד פתוח להרצת מודלי שפה ישירות על המחשב — בלי ענן, בלי רישום, בלי תשלום. עובד על macOS, Windows ו־Linux ומגיע עם קטלוג של מאות מודלים פתוחים.

פרטיות מלאה

הנתונים לא עוזבים את המחשב. השיחות, הקוד והמסמכים — נשארים אצלנו בלבד.

עובד offline

אחרי ההורדה הראשונה — אין צורך בחיבור לאינטרנט. עובד בטיסה, ברכבת, בכל מקום.

API מקומי

חושף REST API בפורמט OpenAI — קל לשלב עם סקריפטים, אפליקציות וכלי פיתוח.

Ollama מריץ LLM ישירות במחשב שלנו.

אפשר להוריד ולהריץ בשתי פקודות (ollama pull ואז ollama run) — או הכל בפקודה אחת עם ollama run בלבד. ב־30 שניות נקבל LLM פרטי שרץ מקומית — בלי ענן, בלי מנוי, בלי הגבלות.

מה זה LLM?

Large Language Model. רשת עצבית עצומה שאומנה על מיליארדי טקסטים מהאינטרנט — ולמדה לחזות את המילה הבאה. מתוך החיזוי הזה נולדת היכולת לכתוב, לתרגם, לסכם ולתכנת.

פרטיות

כל בקשה — חוזים, קוד, סודות — נשלחת לשרת חיצוני. מי שולט במה שקורה שם?

עלות

תשלום לפי tokens. שימוש כבד = חשבונית שמתגלגלת בלי הפסקה.

תלות

ללא אינטרנט — אין AI. rate limits, downtime, ושינויי תמחור של הספק.

01/04

פרטיות מלאה

כל מילה — נשארת במחשב שלנו. אין שליחת מידע לשרת חיצוני. חוזים, קוד מקור, נתוני לקוחות — הכל פרטי. מהמכשיר לא יוצא כלום.

02/04

ללא עלויות

אין tokens, אין חשבונית חודשית, אין rate limits. השקעה אחת בחומרה — ואז שימוש חופשי לנצח. נריץ כל מה שנרצה, מתי שנרצה.

03/04

עובד offline

הורדה ראשונה אחת — וזהו. בטיסה, ברכבת, בכפר באמצע שום מקום. המודל רץ על המכשיר שלנו, בלי תלות בחיבור רשת.

04/04

תגובה מיידית

בלי latency של רשת. המודל רץ ישירות על המעבד או הכרטיס גרפי שלנו. שניות בודדות מהשאלה לתשובה.

RAM · the first law

המודל חייב להיכנס
לזיכרון.

זה הכלל הראשון של מודלים מקומיים. כמות ה־RAM הזמינה במחשב שלנו קובעת אילו מודלים נוכל בכלל להריץ. מודל גדול מהזיכרון = לא ירוץ.

memory = capability

כמה RAM נצטרך?

גודל מודל קטגוריה RAM מינ׳ RAM מומלץ
1B – 3B זעיר · מהיר 4 GB 8 GB
7B – 8B בינוני · יומיומי 8 GB 16 GB
13B – 14B גדול · רציני 16 GB 32 GB
32B – 70B ענק · מקצועי 32 GB 64 GB+

המלצה לרוב המשתמשים: Mac עם 16GB unified memory או PC עם 32GB RAM ו־GPU של 8GB+ — נותן גישה לכל מה שנצטרך ביום־יום.

Ollama בענן · מודלים גדולים בלי חומרה

חלק מהמודלים פשוט גדולים מדי בשביל לרוץ במחשב הביתי. ל־Ollama יש שירות ענן שמריץ אותם בשבילנו — דרך אותו ה־CLI בדיוק.

מודלים ענקיים

גישה למודלים שלא נכנסים בזיכרון המקומי — 70B, 120B ומעלה.

אותו ה־CLI

אותן הפקודות שכבר הכרנו — רק עם תג :cloud בסוף שם המודל.

תוכנית עם מגבלות

נרשמים בחינם ב־ollama.com. יש מכסת שימוש חודשית; ניתן לשדרג בתשלום.
